# clyvn - Bio Link Page

✨ A neon-themed animated bio link page with glowing cursor and music player.

## Features
- ✨ Animated neon background with color gradients
- 🔮 Glowing neon name with smooth animations
- 🖱️ Custom glowing cursor effect
- 🔗 Links to Discord and Instagram
- 🎵 Music player (ready to add your tracks)
- 📱 Fully responsive design
- 🎨 Dark mode with vibrant neon colors

## Links
- **Discord**: peoplearesick
- **Instagram**: @clyvn

## Setup
1. The page is live at: `https://clyvnx3-byte.github.io/`
2. To add a custom domain, update DNS settings to point to GitHub Pages
3. To add music, replace the audio src in `index.html` with your music URL

## Customization
- Edit `style.css` to change colors (use the `:root` variables)
- Modify `index.html` to update links and content
- Add more social links following the same pattern

## Music Setup
To add music:
1. Host your audio file online (or in the repo)
2. Update the `<source src="YOUR_MUSIC_URL">` line in `index.html`
3. The player will be ready to use

Enjoy! 🎉
